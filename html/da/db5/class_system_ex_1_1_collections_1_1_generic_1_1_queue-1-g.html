SystemEx.Collections.Generic.Queue< T > Class Template Reference

A simple fixed‑size FIFO queue implemented as a thin wrapper around a Deque<T>.
Provides enqueue and dequeue operations using the deque’s front/back insertion and removal semantics. More...
