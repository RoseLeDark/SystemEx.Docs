Detailed Description

LinkedNodeIterrator<T> provides STL‑style traversal over intrusive doubly‑linked lists built from LinkedNode<T>. It operates on snapshot clones of nodes to allow safe algorithmic manipulation without mutating the underlying list structure.

The iterator supports bidirectional and offset‑based movement:

  • Forward() – advance by one node
  • Back() – move to the previous node
  • Advance(offset) – move by a signed offset
  • Clone() – create a copy of the iterator at its current position

This type integrates with the SystemEx iterator interfaces via IRandomAccessIterator<T>, making it suitable for generic algorithms such as Distance, Advance, LowerBound and UpperBound.
