SystemEx.BitUtils Class Reference

Provides low‑level bit manipulation utilities for all primitive integer types.
Includes bit extraction, bit setting, bit flipping, bit masks, and bit rotation for signed and unsigned 8‑, 16‑, 32‑ and 64‑bit values. More...
