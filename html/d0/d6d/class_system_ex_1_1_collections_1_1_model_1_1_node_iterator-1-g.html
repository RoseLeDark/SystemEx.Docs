Detailed Description

The iterator advances through a chain of nodes using the Next link. It stops automatically when the last node is reached, determined by Node<T>.HasNext. No sentinel node is used; the last node itself represents the end position.

This iterator is lightweight and allocation-free, intended for engine-level intrusive data structures.
