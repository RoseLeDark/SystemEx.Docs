SystemEx.Device.Memory.DeviceSharedBuffer< TDeviceSharedBackend > Class Template Reference

Represents a shared memory bridge between a DeviceBuffer and a hardware‑specific backend implementing IDeviceSharedBackend.
Manages the lifecycle of a hardware buffer, including creation, upload, download, and cleanup, depending on the configured SharedCacheType. More...

Public Member Functions

bool Begin ()
 Begins a shared memory session by locking the cache and creating the appropriate hardware buffer.
For SharedCacheType.ReadOnly, a read buffer is created.
For SharedCacheType.WriteOnly and SharedCacheType.ReadWrite, the cache contents are uploaded to a write buffer.
void End ()
 Ends the shared memory session by downloading data from the hardware buffer when required and unlocking the cache.
For write‑enabled modes, the backend writes modified data back into the cache.
For read‑only mode, the hardware buffer is simply closed.
