SystemEx.Collections.Generic.SortedMultiMap< T, TU > Class Template Reference

A map that allows multiple entries with the same key.
Unlike Map<T, TU>, which prevents duplicate pairs, a MultiMap<T, TU> accepts all entries without checking for existing keys or values. /// Sorting is performed eagerly whenever elements are added or inserted, depending on the SortedMap<T, TU> setting. More...
