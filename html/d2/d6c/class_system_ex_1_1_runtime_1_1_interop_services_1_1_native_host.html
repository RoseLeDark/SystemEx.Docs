Static Public Member Functions

static ? Module LoadModule (string strPath)
 Loads a native module from the specified path.
If the module has already been loaded previously, the cached instance is returned instead of loading it again.
This prevents duplicate OS handles and improves performance.
static bool FreeModule (Module module)
 Frees a previously loaded module and removes it from the internal cache.
If the module is not currently tracked, the method returns false.
The underlying OS handle is released using Module.Unload(Module).
static ? T GetFunction< T > (string module, string func)
 Resolves a native function from a module identified by its path.
If the module is loaded, the function is resolved and converted into a managed delegate of type T .
If the module is not found, null is returned.
static ? T GetFunction< T > (Module? module, string func)
 Resolves a native function from the specified module and converts the resulting function pointer into a managed delegate of type T .
The conversion is performed using Marshal.GetDelegateForFunctionPointer<T>(IntPtr).

Static Private Attributes

static Map< string, Module > m_loadedModule = new Map<string, Module>()
 Stores all modules loaded through LoadModule(string).
The key is the module path, and the value is the corresponding Module instance.
This cache ensures that each native library is loaded only once.
