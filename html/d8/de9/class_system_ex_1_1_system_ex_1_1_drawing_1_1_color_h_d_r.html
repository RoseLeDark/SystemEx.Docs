SystemEx.SystemEx.Drawing.ColorHDR Class Reference

Represents a high‑dynamic‑range color in the HSV color space, where the value component can exceed the normalized range (0–1) and extend up to MaxValue. Provides hue‑aware interpolation, brightness manipulation, and arithmetic operations for HDR color processing. More...
