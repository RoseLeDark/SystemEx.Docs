Detailed Description

Node<T> represents the minimal building block for forward-only intrusive data structures. It stores a value and one Next pointer.

The class provides C++-style operations commonly found in std::forward_list:

  • PushBack() – append a node at the end
  • InsertAfter() – insert a node directly after this node
  • EmplaceAfter() – construct and insert a node after this node
  • EraseNext() – remove the node after this node

No backward traversal is supported. For bidirectional intrusive lists, use LinkedNode<T>.
