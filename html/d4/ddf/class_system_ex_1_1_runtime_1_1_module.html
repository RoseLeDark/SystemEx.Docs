Package Functions

IntPtr LoadFunc (string func)
 Resolves a function exported by the native module using the active platform backend loader.
The returned pointer is obtained through the OS‑specific implementation of ProcLoader, which internally calls:

  • GetProcAddress on Windows
  • dlsym on Linux
  • dlsym on macOS

The caller is responsible for converting the returned pointer into a managed delegate if required.
