SystemEx.Collections.Generic.MultiMap< TT, TU > Class Template Reference

A map that allows multiple entries with the same key.
Unlike Map<T, TU>, which prevents duplicate pairs, a MultiMap<TT, TU> accepts all entries without checking for existing keys or values. More...
