SystemEx.Device.Intertropt.IDeviceSharedBackend Interface Reference

Defines the low‑level backend interface used by DeviceSharedBuffer and related shared‑memory abstractions.
A backend represents a hardware‑specific buffer implementation such as RAM, OpenCL, Vulkan, CUDA, or any custom device memory.
Each backend is responsible for creating, writing, reading, and closing its own hardware buffer handle. More...
