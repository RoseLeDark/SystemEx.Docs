SystemEx.Collections.Generic.TupleMap Class Reference

A dynamic, list‑backed map structure that stores ITuple objects. Provides tuple‑based lookup, traversal, range operations, and key/value convenience helpers.
This implementation allows arbitrary tuple arity, but assumes that index 0 represents the key and index 1 represents the value. More...
