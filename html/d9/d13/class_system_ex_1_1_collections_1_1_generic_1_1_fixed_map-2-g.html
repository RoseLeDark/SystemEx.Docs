SystemEx.Collections.Generic.FixedMap< T, TU > Class Template Reference

A fixed‑size map storing Pair<T, TU> elements in a contiguous array.
Unlike dynamic maps, a FixedMap<T, TU> never grows: once the capacity is reached, no further insertions are possible. More...
